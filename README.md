## Download

1. Clone this project

```bash
  git clone https://github.com/reynaldomarchell/travel-list.git travel-list
```

2.  Go to the project directory

```bash
  cd travel-list
```

3. Install the dependencies

```bash
  npm install
```

4. Start the app

```bash
  npm start
```
